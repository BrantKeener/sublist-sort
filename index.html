Sublist Sort

Please note that this is being served as a GitHub page without server access.
The data used to build the list looks just like the response data from the database, and can be found in './public/front.js'
Please note that if you wish to use data from a database, a simple mySQL database can be setup with the data from front.js.
Add a .env file, and give it two parameters: process.env.SUBLIST_PASS, and process.env.PORT.
